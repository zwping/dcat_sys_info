## 安装

> 如需使用sys_info/os页面, 请安装`linfo/linfo`

1. `composer require zwping/dcat_sys_info`
2. 扩展中更新&启用
3. 配置用户菜单&权限
4. 打开 http://your-host/admin/sys_info/phpinfo
