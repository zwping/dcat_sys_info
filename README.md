## 安装

1. `composer require zwping/dcat_sys_info`
2. 扩展中更新&启用
3. 配置用户菜单&权限
4. 打开 http://your-host/admin/dcat_sys_info


打开`http://your-host/admin/dcat_sys_info`, 可以看见这些定时任务
